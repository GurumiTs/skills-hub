# Tech Stack Playbooks

此目錄存放語言、框架、Legacy 系統與專案慣例的參考規則。

## 原則

* C#、ASP.NET、Java、JavaScript、SQL Server、Oracle、WebForms 等不應各自做成 Skill。
* 技術棧知識應放在 playbook，讓 Developer Agent 或相關 Skill 在需要時參考。
* Playbook 應描述相容性、版本限制、常見風險、測試方式與不得使用的寫法。
