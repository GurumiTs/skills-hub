# Review Playbooks

此目錄存放 Code Review、設計品質、可維護性、相容性與 regression risk 的參考規則。

## 原則

* Review 結論需區分 blocking 與 non-blocking。
* Code Review 不取代 DB Review 或 Security Review。
* Review 必須給出可執行的修正建議。

## 建議後續文件

| 文件 | 用途 |
|---|---|
| `code-review.md` | Code Review 標準 |
| `legacy-review.md` | Legacy 專案 review 原則 |
| `blocking-issues.md` | Blocking issue 分級規則 |
